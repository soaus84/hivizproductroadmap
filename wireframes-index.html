Pipeline stages — how insights flow
Insight & Incident Pipeline — Prioritise → Learn → Improve → Resolved
The same Kanban structure handles both insights (generated by AI from observation patterns) and incidents (triaged from the incident pipeline). Cards move manually — no gate required. Endorsement adds social weight but never blocks progression.
Prioritise
Endorse for action & triage. AI draft + source observations visible. Safety manager decides whether to act.
Learn
Enquiry dispatched to supervisors. Live synthesis updates as responses arrive. Toolbox talk generated and delivered.
Improve
Corrective actions assigned and tracked. Actions created from enquiry findings. Progress shown on card.
Resolved
All actions complete. Endorsers notified automatically. Archived in Resolved column for 45 days.
